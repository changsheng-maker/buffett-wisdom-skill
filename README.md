## 安装

### 方法一：拖拽安装（最简单）

1. 下载本仓库中的 `buffett-wisdom.skill` 文件
2. 打开 Claude Code
3. 将 `.skill` 文件拖入 Claude Code 窗口
4. 技能自动安装完毕，即可使用

### 方法二：手动安装

```bash
# 克隆仓库
git clone https://github.com/changsheng-maker/buffett-wisdom-skill.git

# 复制到 Claude Code 技能目录
# macOS
cp buffett-wisdom-skill/buffett-wisdom.skill ~/.claude/skills/

# Linux
cp buffett-wisdom-skill/buffett-wisdom.skill ~/.claude/skills/

# Windows - 将文件复制到 %USERPROFILE%\.claude\skills\
```

## 功能

将 Claude 变成巴菲特投资专家，在你的个人 Obsidian Vault 中检索完整资料，回答投资问题。

### 核心功能

| 功能 | 说明 |
|---|---|
| **概念解释** | 深度讲解巴菲特核心概念（护城河/安全边际/内在价值/市场先生/能力圈等） |
| **原话引用** | 检索 1965-2024 年间 60 封股东信中的精确引文 |
| **公司分析** | 用巴菲特框架分析公司（61家公司完整档案） |
| **哲学应用** | 用巴菲特思维分析现代投资决策 |
| **历史脉络** | 追踪巴菲特思想在四个时代的演变（1956-至今） |
| **交叉引用** | 通过丰富的双向链接连接概念、公司与信件 |

### 触发类型

- **概念问题**："什么是护城河"、"安全边际怎么用"
- **哲学问题**："巴菲特怎么看市场波动"、"为什么要长期持有"
- **公司问题**："巴菲特为什么投苹果"、"盖可保险好在哪"
- **历史问题**："1987股灾巴菲特做了什么"、"他和芒格怎么认识的"
- **应用问题**："用巴菲特框架分析X公司"、"这种情况他会怎么选"
- **引用问题**："他说过什么关于护城河的话"、"怎么描述市场先生"

## 工作原理

技能检索你本地 Obsidian Vault 中的巴菲特投资知识库：`~/Documents/Obsidian/花海/04 Knowledge 知识网络/05 Topics 专题领域/09 巴菲特投资/`

### 回答策略

1. **判断问题类型** — 概念、引文、公司、哲学应用、还是历史事件
2. **精准检索** — 先读相关概念卡，再找支持性信件/公司档案
3. **综合输出** — 从以下维度构建答案：
   - 概念定义（用巴菲特原话）
   - 2-3个关键引文
   - 典型案例（盖可保险/可口可乐/华盛顿邮报）
   - 思想演变脉络
   - 相关概念链接

### 回答风格

1. **结论先行** — 一句话回答核心问题
2. **引用原话** — 用巴菲特自己的话
3. **具体数据** — 投资年份、回报率、倍数
4. **反直觉洞察** — 点出多数人忽略的关键
5. **行动收尾** — 告诉用户下一步想什么或做什么
6. **注明出处** — 引用 Vault 文件路径

## 使用示例

### 示例一：护城河概念

**提问：**
```
解释巴菲特的"护城河"概念，用他的话和实际案例说明
```

**技能行为：**
- 读取 `concepts/护城河.md`
- 找到 2007 年股东信中关于持久护城河的引文
- 识别两种类型：低成本（GEICO）和强品牌（可口可乐）
- 提取具体投资回报数据

**预期输出：**
> "护城河是巴菲特最核心的投资概念之一。2007年他在股东信中说：'一道需要不断重建的护城河，其实根本没有护城河。'他将护城河分为两种类型：低成本优势（如GEICO保险）和强品牌（如可口可乐）。GEICO的市场份额从2.5%增长到12%，正是护城河威力的证明。"
> — [[2007-巴菲特致股东信]]

### 示例二：市场先生

**提问：**
```
巴菲特怎么看市场波动？市场先生这个比喻是什么意思？
```

- 找到 1987 年股东信中"市场是你的仆人"原话

- 给出 1973 年华盛顿邮报案例

**预期输出：**
> "市场先生来自格雷厄姆的《聪明的投资者》，不是巴菲特的发明，但巴菲特最爱用这个比喻。他说：'市场先生是来为你服务的，不是来引导你的。对你有用的是他的钱包，而不是他的智慧。'1973年华盛顿邮报大跌时，巴菲特看透了市场先生的情绪，反而大举买入。"
> — [[1987-巴菲特致股东信]]

### 示例三：苹果投资分析

**提问：**
```

**预期输出：**
> "巴菲特投资苹果的核心逻辑是护城河。他看中的是苹果生态系统的忠诚度——用户一旦进入苹果生态，转换成本极高。2018年他在股东信中提到：'苹果产品的受欢迎程度令人难以置信，特别是对用户粘性的贡献。'这不是普通消费品的护城河，而是生态系统的护城河。"
> — [[2018-巴菲特致股东信]]

### 示例六：芒格的影响

**提问：**
```
巴菲特和芒格是什么关系？芒格如何影响巴菲特的投资哲学？

**预期输出：**

### 示例八：概念演变

**提问：**
```
巴菲特的"内在价值"概念是怎么演变的？
```

**技能行为：**
- 读取 `concepts/内在价值.md`
- 追踪四个时代（合伙人期1956-1969、转变期1970-1989、成熟期1990-2009、现代2010-至今）的演变

**预期输出：**
